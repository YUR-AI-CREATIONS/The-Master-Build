What is a Dockerfile?
ChatGPT said:

A Dockerfile is a script that tells Docker
 how to build a containerized environment for your application. Think of it as a blueprint for packaging your app (like LeadVault.OS) into a self-contained, portable box that runs the same on any machine — laptop, server, or cloud.

🧱 What It Contains:

A Dockerfile usually includes:

Instruction	What It Does
FROM	Specifies the base image (e.g., Python, Ubuntu)
WORKDIR	Sets the working directory inside the container
COPY	Adds your local files to the container
RUN	Installs dependencies (e.g., pip install)
CMD / ENTRYPOINT	Specifies the command to run when the container starts
🔧 Example: Dockerfile for LeadVault.OS
# 1. Base Image
FROM python:3.11-slim

# 2. Set working directory
WORKDIR /app

# 3. Copy files
COPY . .

# 4. Install dependencies
RUN pip install --no-cache-dir -r requirements.txt

# 5. Run the enrichment engine
CMD ["python", "leadvault_enrichment_engine.py"]

🛰️ Why Use It?

Portability: Run the same stack anywhere (Mac, Windows, cloud).

Isolation: Your app won't break due to local system conflicts.

Deployment-ready: Combine with docker-compose.yml to launch Postgres, enrichment service, and frontend with one command.

    try:

What is a Dockerfile?
ChatGPT said:

A Dockerfile is a script that tells Docker
 how to build a containerized environment for your application. Think of it as a blueprint for packaging your app (like LeadVault.OS) into a self-contained, portable box that runs the same on any machine — laptop, server, or cloud.

🧱 What It Contains:

A Dockerfile usually includes:

Instruction	What It Does
FROM	Specifies the base image (e.g., Python, Ubuntu)
WORKDIR	Sets the working directory inside the container
COPY	Adds your local files to the container
RUN	Installs dependencies (e.g., pip install)
CMD / ENTRYPOINT	Specifies the command to run when the container starts
🔧 Example: Dockerfile for LeadVault.OS
# 1. Base Image
FROM python:3.11-slim

# 2. Set working directory
WORKDIR /app

# 3. Copy files
COPY . .

# 4. Install dependencies
RUN pip install --no-cache-dir -r requirements.txt

# 5. Run the enrichment engine
CMD ["python", "leadvault_enrichment_engine.py"]

🛰️ Why Use It?

Portability: Run the same stack anywhere (Mac, Windows, cloud).

Isolation: Your app won't break due to local system conflicts.

Deployment-ready: Combine with docker-compose.yml to launch Postgres, enrichment service, and frontend with one command.

import React, { useEffect, useState } from "react";
import axios from "axios";
import { Card, CardContent } from "@/components/ui/card";
import { Button } from "@/components/ui/button";
import { Input } from "@/components/ui/input";


export default function LeadVaultDashboard() {
  const [leads, setLeads] = useState([]);
  const [filter, setFilter] = useState(0);
  const [exporting, setExporting] = useState(false);


  useEffect(() => {
    fetchLeads();
  }, []);


  async function fetchLeads() {
    const response = await axios.get("/api/leads");
    setLeads(response.data);
  }


  async function exportCSV() {
    setExporting(true);
    const response = await axios.get("/api/export/csv", { responseType: "blob" });
    const blob = new Blob([response.data], { type: "text/csv" });
    const url = window.URL.createObjectURL(blob);
    const link = document.createElement("a");
    link.href = url;
    link.setAttribute("download", "leads.csv");
    document.body.appendChild(link);
    link.click();
    link.remove();
    setExporting(false);
  }


  const filtered = leads.filter(lead => lead.score >= filter);


  return (
    <div className="p-6 space-y-4">
      <h1 className="text-3xl font-bold">LeadVault.OS Dashboard</h1>


      <div className="flex items-center gap-4">
        <Input
          type="number"
          min={0}
          max={10}
          value={filter}
          onChange={(e) => setFilter(Number(e.target.value))}
          placeholder="Min Score Filter (0-10)"
        />
        <Button onClick={exportCSV} disabled={exporting}>
          {exporting ? "Exporting..." : "Export CSV"}
        </Button>
      </div>


      <div className="grid grid-cols-1 md:grid-cols-3 gap-4">
        {filtered.map((lead, i) => (
          <Card key={i} className="bg-white border border-gray-200 shadow">
            <CardContent className="p-4">
              <p className="font-semibold text-lg">{lead.business_name}</p>
              <p className="text-sm text-gray-500">{lead.website}</p>
              <p className="text-sm">Score: {lead.score}</p>
              <p className="text-sm">Tags: {lead.tags?.join(", ")}</p>
              <Button className="mt-2" size="sm" variant="secondary" onClick={() => handleOutreach(lead)}>Trigger Outreach</Button>
            </CardContent>
          </Card>
        ))}
      </div>
    </div>
  );
}

import os
from fastapi import APIRouter, HTTPException
from pydantic import BaseModel, EmailStr
from sendgrid import SendGridAPIClient
from sendgrid.helpers.mail import Mail


router = APIRouter()


SENDGRID_API_KEY = os.getenv("SENDGRID_API_KEY")
FROM_EMAIL = os.getenv("FROM_EMAIL", "outreach@leadvault.ai")


class EmailPayload(BaseModel):
    to_email: EmailStr
    subject: str
    message: str


@router.post("/api/send-email")
async def send_email(payload: EmailPayload):
    if not SENDGRID_API_KEY:
        raise HTTPException(status_code=500, detail="Missing SendGrid API Key")


    message = Mail(
        from_email=FROM_EMAIL,
        to_emails=payload.to_email,
        subject=payload.subject,
        html_content=payload.message
    )


    try:
        sg = SendGridAPIClient(SENDGRID_API_KEY)
        response = sg.send(message)
        return {
            "status": "sent",
            "to": payload.to_email,
            "code": response.status_code
        }
    except Exception as e:
        raise HTTPException(status_code=500, detail=f"Email send failed: {str(e)}")
